# NayePankh-foundation
A simple and responsive website created using HTML and CSS for the NayePankh Foundation.

## Features
- Responsive Design
- Hero Section
- About Us Section
- Mission Cards
- Image Gallery
- Contact Section
- Hover Effects

## Technologies Used
- HTML5
- CSS3

## Purpose
This project was created to showcase the mission and activities of NayePankh Foundation while demonstrating frontend web development skills.
